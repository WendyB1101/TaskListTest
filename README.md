# Task Manager — Laravel CRUD

Простое приложение для управления задачами, разработанное на Laravel 12.

## Функционал

- Список задач с пагинацией (10 на страницу)
- Поиск по названию задачи
- Фильтрация по статусу
- Создание, редактирование, просмотр и удаление задач
- Быстрое изменение статуса прямо со страницы просмотра
- Валидация форм на стороне сервера

## Статусы задач

| Ключ          | Название   |
|---------------|------------|
| `new`         | Новая      |
| `in_progress` | В работе   |
| `done`        | Выполнена  |

## Структура задачи

| Поле          | Тип      | Описание              |
|---------------|----------|-----------------------|
| `id`          | integer  | Первичный ключ        |
| `title`       | string   | Название задачи       |
| `description` | text     | Описание (необязательно) |
| `status`      | enum     | Статус задачи         |
| `created_at`  | datetime | Дата создания         |
| `updated_at`  | datetime | Дата обновления       |

## Установка и запуск

```bash
# 1. Установить зависимости
composer install

# 2. Скопировать .env
cp .env.example .env

# 3. Сгенерировать ключ приложения
php artisan key:generate

# 4. Создать SQLite базу данных
touch database/database.sqlite

# 5. Запустить миграции
php artisan migrate

# 6. (Опционально) Заполнить тестовыми данными
php artisan db:seed

# 7. Запустить сервер
php artisan serve
```

Приложение будет доступно по адресу: http://localhost:8000
